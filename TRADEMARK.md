# Trademark Policy

The project name `metactl`, logos, compatibility badges, and release names identify this project and its releases.

You may:

- use the name to refer to the project;
- state that your tool integrates with `metactl`;
- fork the code under the license terms.

You may not:

- imply that a fork is an official release;
- use a compatibility badge unless the published conformance rules pass;
- use the project name or marks in a way that confuses users about source, maintainership, or support.

Compatibility language should be factual, for example: "tested with metactl 0.1.0".
