# metactl Starter Library

Small public example library for the first metactl alpha release.

- 3 durable roles
- 3 durable policies
- 6 supported targets (claude-code, codex-cli, cursor, filesystem-agent, gemini-cli, openclaw)
- 11 promoted example packs with provenance and validation evidence

Highlighted public packs: `metactl-project-onboarding` helps agents install metactl safely in greenfield and brownfield repos, `agent-candidate-library-installer` sets up private or local agent-authored candidate libraries with source registration and hook verification, `python-refactor` demonstrates multi-surface instruction projection, `agentic-artifact-forge` turns prompts and instructions into preservation-first skills, commands, and rules, and `library-organization-guide` acts as a stewardship skill for organizing packs, profiles, and config-layer decisions.

This library is intentionally small. It exists to prove the portable kernel, not to act as a marketplace.
