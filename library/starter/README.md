# metactl Starter Library

Small public example library for the first metactl alpha release.

- 3 durable roles
- 3 durable policies
- 5 supported targets (claude-code, codex-cli, cursor, gemini-cli, openclaw)
- 8 promoted example packs with provenance and validation evidence

Highlighted public packs: `metactl-project-onboarding` helps agents install metactl safely in greenfield and brownfield repos, `python-refactor` demonstrates multi-surface instruction projection, and `library-organization-guide` acts as a stewardship skill for organizing packs, profiles, and config-layer decisions.

This library is intentionally small. It exists to prove the portable kernel, not to act as a marketplace.
